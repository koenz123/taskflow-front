### UI Design Spec (TaskFlow) — минималистичный стиль

Этот документ фиксирует текущий дизайн-системный подход проекта, чтобы новые экраны выглядели единообразно.

## Базовые принципы

- **Минимализм**: без “карточек” с тяжелыми тенями/градиентами. Акцент — на типографике, сетке и воздухе.
- **Один фон**: общий фон приложения — почти черный. Внутренние блоки не должны иметь собственных “плашек”, если это не нужно для читабельности.
- **Разделители вместо рамок**: предпочтительнее `border-top`/`border-bottom` тонкой линией `rgba(255,255,255,0.06)`, чем рамки вокруг каждого блока.
- **Hover**: вместо “подпрыгивания” — **`transform: scale(1.03)`** и легкое изменение `border/background`.
- **Резиновость**: размеры и отступы через `clamp()` и `min()/max()`, чтобы UI выглядел одинаково на разных экранах.

## Цвета (токены)

Используем CSS-переменные из `src/index.css`:
- `--bg0`, `--bg1`: фон (почти черный)
- `--text`: основной текст (почти белый)
- `--muted`, `--muted2`: вторичный текст (серый)
- `--border`, `--border2`: линии/разделители
- `--accent`, `--accent2`: акцент (для активных состояний/бейджей)

Рекомендуемые альфы:
- разделитель: `rgba(255,255,255,0.06)`
- легкая плашка hover: `rgba(255,255,255,0.04–0.06)`
- вторичный текст: `rgba(255,255,255,0.55–0.70)`

## Типографика

- Заголовок страницы: `clamp(22px, 2.2vw, 30px)`, `font-weight: 700`
- Заголовок секции: `14–16px`, `font-weight: 600`
- Основной текст: `14–15px`, `line-height: 1.5–1.7`
- Лейблы полей: `12–13px`, `opacity: 0.7–0.9`, `text-transform: uppercase` (опционально)

## Отступы / сетка

- Общие паддинги страницы: `clamp(16px, 2.4vw, 28px)`
- Gap между крупными блоками: `clamp(16px, 2.2vw, 24px)`
- Gap внутри форм: `10–14px`

## Компоненты

### Кнопки

- Высота: `44px` для кнопок в строке с инпутом (как Filters)
- Радиус: `10–12px`
- По умолчанию: без градиентов, фон `rgba(255,255,255,0.03)`
- Hover:
  - `transform: scale(1.03)`
  - `background: rgba(255,255,255,0.05–0.06)`
  - `border-color` чуть ярче

### Инпуты / textarea

- `border: 1px solid rgba(255,255,255,0.12)`
- `background: rgba(255,255,255,0.02–0.04)` или `rgba(0,0,0,0.12)` (но единообразно внутри экрана)
- `border-radius: 12–14px`
- Focus: тонкий ring через `outline`/`box-shadow` (с `--ring`)

### Списки / строки

- Строка кликабельна целиком (как карточка задания)
- Hover: легкая подсветка строки (без “плашек” вокруг всего блока)

### Бейджи статуса

- Цветной фон с альфой (open — зеленый, in_progress — синий, closed — серый)
- Текст в upper-case, маленький размер `11px`

## Анимации

- Переходы: `200–420ms`, easing: `cubic-bezier(0.2, 0.8, 0.2, 1)` для раскрытий
- Для micro-interactions: `200ms ease`

## Адаптив

- Desktop: двухколоночные grid где уместно
- Mobile: все в 1 колонку; кнопки в строке превращаются в колонку
- Стремимся к `clamp()` вместо фиксированных px где это влияет на “ощущение” интерфейса
